## Build

```bash
cabal build
```

## Run CLI Game

```bash
cabal run -- -m cli
```

## CLI Commands

| Key | Command     |
|:----|:------------|
| Esc | Exit        |
| f   | Place Stone |
| g   | Pass        |
| k   | Move top    |
| l   | Move right  |
| j   | Move down   |
| h   | Move left   |


## OS Support

Only UNIX and UNIX-like operating systems are supported.
